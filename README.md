# Spring-Project
스프링 + JSP 파싱_ 음악 구인 구직 커뮤니티 사이트

# 🖥️ 프로젝트 소개
클래식 연주자들이 정보를 파악하는 커뮤니티 사이트
단톡방에서 벗어나 구인, 홍보, 커뮤니티를 한눈에 볼 수 있는 사이트
